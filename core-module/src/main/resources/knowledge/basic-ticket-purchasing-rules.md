# 购票基本规则

## 一、可购买的车票类型

在12306.cn网站，可以购买铁路客票系统发售的全价票、儿童优惠票、学生优惠票、残疾军人票（持中华人民共和国残疾军人证、中华人民共和国伤残人民警察证、国家综合性消防救援队伍残疾人员证的人员）四种各车次、各席别的车票[citation:9]。

## 二、有效身份证件

- 居民身份证（含临时）
- 护照
- 港澳居民居住证
- 台湾居民居住证
- 港澳居民来往内地通行证
- 台湾居民来往大陆通行证
- 外国人永久居留身份证
- 外国人护照

## 三、订单限制

1. 一张有效身份证件同一乘车日期同一车次只能购买一张车票（高铁动卧列车除外）[citation:9]
2. 一笔订单不能超过19张票[citation:9]
3. 一天内未支付或取消订单合计超过3次，当日限制购票（次日服务时间恢复）[citation:9]

## 四、购票成功确认

购票成功后，系统跳转到"订票成功"页面，订单信息自动保存在"我的订单"中，同时向注册邮箱发送邮件通知、向通知设置中的渠道发送通知信息[citation:9]。

## 五、信息修改

1. **姓名错误**：购票后不能更改姓名，需退票重购[citation:9]
2. **日期、发站、到站、车次、席别错误**：可按规定办理改签、变更到站、退票[citation:9]
3. **身份证件号码错误**：需退票重购[citation:14]
