# 12306火车票报销规则（2025年10月1日起）

## 一、核心变化：全面使用电子发票

根据《国家税务总局 财政部 中国国家铁路集团有限公司公告2024年第8号》，自**2025年10月1日**起，全国铁路旅客运输领域**全面使用电子发票（铁路电子客票），不再提供纸质报销凭证**。

### 过渡期安排
| 乘车日期          | 报销凭证类型                    |
|---------------|---------------------------|
| 2025年9月30日及以前 | 可开具纸质报销凭证（需在乘车之日起180日内打印） |
| 2025年10月1日及以后 | 统一使用电子发票，不再提供纸质报销凭证       |


## 二、电子发票开具

### 1. 开具时限
旅客可在以下业务办理之日起**180日内**申请开具电子发票：
- 行程结束
- 退票业务办理完成
- 改签业务办理完成

### 2. 开具渠道
可通过以下渠道申请开具：
| 渠道 | 说明 |
|------|------|
| 铁路12306网站/APP | 在线申请，最便捷 |
| 车站售票窗口 | 提供"扫码开票单"，用12306 APP扫码补全信息后开具 |
| 自动售票机 | 自行打印"扫码开票单"，用12306 APP扫码开具 |

### 3. 开具流程（12306 APP）
1. 登录12306账户，进入【我的】页面
2. 点击【电子发票】功能模块
3. 选择需要开具发票的行程订单
4. 填写发票抬头信息（公司全称、纳税人识别号）
5. 提交后下载PDF或OFD格式电子发票

### 4. 可开具的内容
- 已购车票的票款
- 退票费
- 改签费


## 三、电子发票使用与管理

### 1. 查询与下载
开具电子发票后，可通过以下渠道查询、下载：
- 铁路12306（网站/APP）
- 个人所得税APP
- 全国增值税发票查验平台（查验真伪）

### 2. 换开规则
如发生填写发票信息有误或企业信息变更等情况，在**原电子发票开具时限内**可申请换开，最多可换开**3次**。

### 3. 电子发票效力
- 铁路电子发票属于**增值税电子普通发票**，票面税率为9%，可作为**进项税抵扣**凭证
- 与纸质报销凭证具有同等法律效力，**不可重复开具**[citation:6]
- 电子发票可**多次下载、打印**，但法律效力以电子文件为准，需保存原PDF/OFD电子文件

### 4. 会计归档要求
- 电子发票原件（PDF/OFD+XML）需保存**10年**（《会计档案管理办法》规定）
- 如以电子发票的纸质打印件入账，必须**同时保存**该打印件对应的电子会计凭证
- 建议通过财务系统对"发票号码+校验码"进行**查重**处理，防止重复报销


## 四、特殊情形

### 1. 购票人（代办人）代开
购票人（代办人）可为乘车人申请开具电子发票：
- 凭乘车人有效身份证件，或凭订单号+购票人有效身份证件
- 到车站售票窗口申请，获取"扫码开票单"后通过12306 APP开具

### 2. 团体票
旅游、学生、研学等**团体票不支持乘车人本人开具**，需由购票人按相关流程开具电子发票。

### 3. 暂不支持开具电子发票的情形
以下情况继续沿用现行铁路报销凭证：
- 非实名制车票
- 应急纸质车票
- 中铁银通卡/E卡通车票
- 跨境旅客运输车票

### 4. 行程信息提示单
"行程信息提示单"仅供乘车提醒，包含车次、日期、座位等信息，但**不能作为报销依据**。
