# 学生票规则

## 一、适用人群

在全日制高等学校、科研院所、军事院校、普通中小学、中等职业学校就读，家庭居住地和学校所在地不在同一城市的在校学生。

## 二、购买条件

1. 家和学校不在同一城市
2. 出行目的为往返家庭与学校
3. 中小学生须持学校开具的、加盖公章的书面证明到车站窗口购买，线上无法购买

## 三、资质核验

符合购买学生优惠票条件的旅客，应到车站指定售票窗口或自动售票机办理一次本人居民身份证件与学生优惠卡的核验手续。学生优惠票需**每学年乘车前办理一次**核验手续。

通过核验手续的旅客购票后可凭居民身份证件自助办理实名制验证和进出站检票[citation:9]。

## 四、购票次数

每学年可购买4次单程学生优惠票[citation:12]。

## 五、学生票票价

学生票享受以下优惠票价：
- 普速列车硬座客票、加快票、空调票：公布票价的50%
- 动车组二等座/二等包座/多功能座/一等座/动车组卧铺：全价执行票价的75%

## 六、优惠区间

优惠区间为"家庭所在地—学校所在地"。如需更改优惠区间，需找学校开具证明并到火车站办理，APP上无法修改。

## 七、2025年寒假期间特殊调整（2025年1月1日至3月31日）

学生旅客可根据与家人团聚的实际地点，在铁路12306系统中灵活设置学生票优惠区间，并购买相应的学生优惠票。乘车途中工作人员仅核验铁路12306系统学生优惠资质信息中的学校所在地与学生证中的学校所在地是否一致。
