# 儿童票规则

## 一、购票标准（按年龄，不以身高为标准）

| 年龄条件             | 购票规则                                  |
|------------------|---------------------------------------|
| 未满6周岁（6岁生日前）     | 可免费乘车，不单独占用席位，需由成人携带并提前申报[citation:3] |
| 年满6周岁且未满14周岁     | 购买儿童优惠票（半价）[citation:3]               |
| 年满14周岁（14岁生日当天起） | 购买全价票[citation:3]                     |

## 二、免费儿童乘车规定

每一名持票成年人旅客可免费携带一名未满6周岁且不单独占用席位的儿童乘车。超过一名时，超过人数应当购买儿童优惠票[citation:8]。

## 三、免费儿童申报方式

成人购票后，进入"本人车票"页面，通过"免费乘车儿童申报"入口关联孩子证件信息。成人票改签退票时，关联的免费儿童信息会自动同步[citation:3]。

## 四、儿童票票价

儿童优惠票票价为成人全价票价的50%[citation:13]。

## 五、乘车要求

- 儿童票乘车人必须持购票时所用的有效身份证件原件进站乘车
- 电子临时身份证明不适用于儿童[citation:13]
- 儿童单独出行不得购买儿童票，必须由成人陪同[citation:13]
